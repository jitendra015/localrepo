Learning Path

Beginner checklist

  1. Create a repository and connect it with GitHub.
  2. Build the page using clean HTML.
  3. Style the page with responsive CSS.
  4. Write a helpful README file.
  5. Commit changes with clear messages.
  6. Push updates to GitHub and publish with GitHub Pages.
